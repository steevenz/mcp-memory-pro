# Domain: Setup

## Purpose
This domain covers the installation, configuration, and integration of the `mcp-memory-pro` server into various development environments.

## Scope
Included in this domain:
- Installation prerequisites (Node.js, NPM).
- Build processes (`npm install`, `npx tsc`).
- IDE-specific integration (Cursor, Trae, Claude Desktop).

Excluded from this domain:
- Core architectural logic (see `Architecture` domain).
- AI behavioral rules (see `Governance` domain).

## Usage
Developers should reference this domain when setting up a new server instance or troubleshooting integration issues between the IDE and the MCP server.
